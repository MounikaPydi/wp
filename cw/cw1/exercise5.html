1. Advising the School

To maintain quality control, the school should enforce a common stylesheet with predefined styles for fonts, colors, and layout. Teachers should update only the content, not the design.

2. Maintaining Consistent Styling

Using an external CSS file ensures a uniform look across all pages, avoiding inconsistencies that would arise if each teacher applied their own styles.

3. HTML Tags & CSS Styles

Some essential HTML tags and CSS styles include:

  • <div> - Defines a container
  • <p> - Represents a paragraph
  • <h1> to <h6> - Headings
  • <table> - Creates tables
  • color, background-color - CSS for styling text
